Welcome to the **chmonitor** documentation.

chmonitor is a simple ClickHouse UI that relies on system tables to help monitor
and provide an overview of your cluster.

This site is built with [Astro Starlight](https://starlight.astro.build). The
full documentation content is synced from the dashboard docs in a follow-up
change.
