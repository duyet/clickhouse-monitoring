## LTS versions

Long-term support (LTS) releases — currently 23.3, 23.8, 24.3, 24.8 — are the recommended deployment targets.
They receive patches for a longer period from the ClickHouse team and are the versions exercised in CI.

Supported major version series: 23, 24, 25, 26.
