# Skills

This directory contains [Agent Skills](https://agentskills.io) for the ClickHouse Monitor dashboard.

## Available Skills

### clickhouse-monitor

**Path:** [`clickhouse-monitor/`](./clickhouse-monitor/)

Specialized knowledge for the ClickHouse Monitor dashboard - a real-time monitoring and observability tool for ClickHouse clusters.

**Features:**
- Dashboard navigation and page reference (45 pages)
- Complete API documentation (58 charts, 63 table endpoints)
- Query monitoring and performance analysis
- Table management and replication
- Merge operations and mutations
- System metrics and monitoring
- Security and logs
- Development patterns and conventions
- ClickHouse version compatibility

**Installation:**
```bash
npx skills add https://github.com/duyet/clickhouse-monitor --path skills/clickhouse-monitor
```

**Usage:**
Use this skill when working with:
- ClickHouse monitoring dashboards
- Query performance analysis
- Database schema exploration
- ClickHouse system table queries
- Dashboard feature development
- API integration with ClickHouse Monitor

**Documentation:**
- [SKILL.md](./clickhouse-monitor/SKILL.md) - Main skill file with quick reference
- [README](./clickhouse-monitor/README.md) - Installation and usage guide
- [API List](./clickhouse-monitor/references/api-list.md) - All endpoints
- [Dashboard Pages](./clickhouse-monitor/references/dashboard-pages.md) - Page reference

- [Development Guide](./clickhouse-monitor/references/development.md) - Contribution patterns

## About Agent Skills

Agent Skills are folders of instructions, scripts, and resources that AI agents can discover and use to do things more accurately and efficiently. This open standard was originally developed by Anthropic.
