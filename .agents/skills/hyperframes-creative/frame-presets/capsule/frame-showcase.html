Rules

Frame Rules

Do
  • Make every text container a pill — 9999px small, 2rem for cards.
  • Wrap every pill, card, and icon in the 2px ink outline.
  • Set Bodoni headlines in ink, sentence case; color lives on stats + pill fills.
  • Use soft offset shadows (4/6/8/12px) in 8% ink — never blur.
  • Float 5–8 candy pills as wallpaper on declarative frames.
Don't
  • No sharp-cornered text containers; no unstroked pills.
  • No colored Bodoni headlines; no uppercase Bodoni.
  • No blurred shadows; no tenth accent color.
  • No shadows on decorative floating pills (atmosphere is flat).
  • Don't blow a headline edge-to-edge — fit to measure.
