05

frame rules

INK ON FIRE

do

  • Set every Barlow display in lowercase weight 900, negative-tracked.
  • Make orange the full environment on declarative frames, the lone accent on dark.
  • Keep chrome in IBM Plex Mono uppercase, 0.14em tracked.
  • Cap bullet lists at three; one statement per frame.
  • Build hierarchy from weight, size, and 1px hairlines only.

don't

  • Never uppercase Barlow display — lowercase is the identity.
  • No second accent color; no cream text on orange (ink on fire).
  • No drop shadows, rounded surfaces, or gradient grounds.
  • No serif companion; chrome is never set in Barlow.
  • Don't pack two display moments into one frame.
