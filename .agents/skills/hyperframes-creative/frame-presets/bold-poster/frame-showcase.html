Frame Rules

One Red · Square Corners

Do

  • Stack hero titles in 3 Shrikhand lines — at least one tilted, one red.
  • Make every numeral red Shrikhand; tilt statement display −5° to −6°.
  • Set eyebrows in Space Grotesk 600 uppercase, 2–3px tracked, red.
  • Build data grids with 3px outer + 1.5px inner double borders.
  • Use red em-dash bullets, capped at three; serif body at line 1.75.

Don't

  • No second accent color; no rounded corners (square only).
  • No drop shadows — the stacked text-shadow on red is the only one.
  • No font substitutes; no Shrikhand body, no Baskerville labels.
  • No default disc bullets; no untilted red statement display.
  • Don't crowd a statement frame — reserve the negative space.
