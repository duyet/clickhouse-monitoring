Frame Rules

Single Accent

Do

  • Start every frame on warm cream; cobalt carries every accent.
  • Set headlines near-black with −0.02em tracking; eyebrows cobalt, uppercase, 0.08em.
  • Render every numeral in cobalt Space Grotesk 600–700.
  • Use tinted cards — 4% fill, 20% border, 10–14px radius; soft, never outlined.
  • Pill chrome at 100px; body in Inter 400, muted gray, line 1.6.

Don't

  • No second accent color; no cobalt headlines.
  • No drop shadows (save the one soft cobalt CTA hover).
  • No square corners; no opaque cobalt borders.
  • No font substitutes; no uppercase body.
  • Don't fill space with heavier borders — add substance, not noise.
